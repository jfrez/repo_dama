# Operacion y continuidad

Runbooks para mantener operativos los pipelines y planificar recuperacion.

Incluye:
- Calendario de ejecuciones y SLA por proceso (ingesta, integracion, reportes) con responsable.
- Indicadores monitoreados (filas, tiempos, fallas) y alertas configuradas.
- Procedimientos de reproceso y recuperacion (puntos de reinicio, lookback permitido, decisiones manuales).
- Dependencias externas (APIs, credenciales, variables de entorno) y contactos de soporte.
- Ensayos de continuidad y fechas de revision.

Enlaza salidas de `data_03_operacion/monitoreo/` y logs guardados en `tmp/` como evidencia.
