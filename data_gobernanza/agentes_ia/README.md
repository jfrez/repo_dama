# Agentes IA de apoyo (demo educacional)

En esta carpeta se agrupan agentes listos para asistir la gobernanza de datos. Cada subdirectorio incluye `Agents.md`, plantillas de ejemplo y definiciones de uso:

- `diccionario/`: curación de diccionarios/glosarios.
- `dq/`: monitoreo y acciones sobre calidad de datos.
- `catalogo/`: fichas de datasets y metadata operativa.
- `linaje/`: documentación de flujos y linaje técnico/funcional.

Carga los prompts en `codex-cli` o en tu herramienta favorita; cada agente describe entradas, salidas y checklist de gobierno que debe aplicar. Ajusta las plantillas según tus políticas locales.
