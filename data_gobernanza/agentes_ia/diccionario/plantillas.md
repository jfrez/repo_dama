# Plantillas de interacción (Curador de diccionario)

## Solicitud para agregar tabla
```
Actúa como Curador de Diccionario.
Entrada: código de tabla <tabla> en <ruta>.
Revisa data_gobernanza/diccionarios/tablas.md y genera una fila Markdown con:
- Tabla | Descripción | Campos clave | Periodicidad | Origen
Incluye nota de propietario y fecha sugerida.
```

## Revisión de consistencia
```
Revisa la tabla <tabla> en <ruta> y compárala con el diccionario.
Indica campos faltantes o con tipo incoherente y sugiere corrección breve.
Checklist: clave, descripción clara, frecuencia, origen, sensibilidad.
```
