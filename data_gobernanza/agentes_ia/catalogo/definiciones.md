# Campos obligatorios en fichas de metadata

- Conjunto: nombre corto y amigable (sin guiones bajos).
- Descripción: qué contiene y para qué se usa.
- Fuente: tipo y ubicación (CSV local, API URL/fixture, tabla previa).
- Frecuencia: batch inicial, continua (definir lookback), cron si aplica.
- SLA: tiempo máximo de frescura aceptable.
- Criticidad: alta (impacta reporting clave), media, baja.
- Sensibilidad: PII/Sensible/No sensible; detallar si hay emails u otros datos personales.
- Responsable: rol y contacto (correo/alias).
