# Gobierno y politicas

Guia para registrar roles, controles y decisiones de acceso del demo educacional alineados con DAMA.

Que documentar:
- Roles y responsables por dominio (Data Owner, Steward, Eng) y ruta de escalacion.
- Matriz de acceso y clasificacion (publico/interno/restringido) con criterios de aprobacion.
- Excepciones otorgadas (fecha, solicitante, vigencia) y justificacion.
- Politicas de retencion y uso aceptable vinculadas a las fuentes/productos registrados en `metadata/`.
- Evidencias de revisiones periodicas y cambios a SLA o contratos de datos.

Usa tablas breves y vincula referencias a `metadata/fuentes/` y `metadata/productos/` para mantener coherencia.
