# Diccionario capa dm

Define los productos de datos y datamarts expuestos a consumidores.

Incluye por producto:
- Layout de columnas (nombre, tipo, definicion de metrica, agregaciones aplicadas).
- Dimensiones y filtros obligatorios, claves de negocio y granularidad.
- SLA de frescura/disponibilidad y consumidores principales.
- Sensibilidad, controles de acceso y retencion asociada.

Plantilla base: `../../plantillas/diccionario.md` complementada con contratos de salida en `../../metadata/productos/`.
