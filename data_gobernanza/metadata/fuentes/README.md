# Metadata de fuentes

Describe cada origen de datos y su contrato de entrega.

Incluye:
- Proveedor o sistema origen, endpoint/ruta, formato y version del contrato.
- Periodicidad y horario de entrega, ventana de lookback y SLA de ingesta.
- Responsable/contacto y canal de soporte para incidentes.
- Clasificacion de datos y controles de acceso o mascaras aplicados en origen.

Vincula el linaje en `../../linaje/` y los diccionarios raw en `../../diccionarios/raw/`.
