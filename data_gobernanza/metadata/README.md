# Metadata y catalogo

Catalogo central de fuentes y productos con SLA, periodicidad y responsables.

Estructura:
- `fuentes/`: fichas de origen/ingesta y contratos de entrada.
- `productos/`: fichas de salida, contratos de consumo y requisitos de acceso.

Cada ficha debe incluir periodicidad, medio de entrega, SLA, contacto, criticidad, retencion, ubicacion y linaje. Usa `../plantillas/catalogo.md` como base y vincula con diccionarios y reglas DQ correspondientes.
