# Metadata de productos y consumos

Documenta los productos de datos expuestos (dm, reportes, APIs, CSV).

Incluye:
- Proposito, consumidor principal, owner/steward y canal de soporte.
- SLA de frescura/disponibilidad, medio de publicacion (CSV, API, dashboard) y ubicacion.
- Layout o contrato de salida (columnas, tipos, definiciones de metricas y filtros).
- Retencion, clasificacion y controles de acceso asociados.
- Dependencias aguas arriba (core/stage) y alertas aplicables en `../../dq_reglas/consumo/`.

Usa `../../plantillas/catalogo.md` para mantener consistencia.
